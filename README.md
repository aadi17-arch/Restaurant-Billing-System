# Restaurant Billing System

A Java desktop application that manages restaurant orders 
and generates bills with database integration.

## 🛠 Tech Stack
- Java
- Java Swing (GUI)
- MySQL / File Database

## ✨ Features
- Add and manage menu items
- Take customer orders through GUI
- Calculate and generate bills automatically
- Database integration for order storage

## 🚀 How to Run
1. Clone the repo
2. Open in Eclipse or IntelliJ
3. Set up the database using database.txt
4. Run the main class from src/com/restaurant
